# Flow – Quotation → Sales Note

## Story
Customer sends list, the nursery returns prices, then the customer confirms.

## Steps
1. Operator creates Quotation with items.
2. Quotation document is shared (export).
3. Customer confirms.
4. Operator converts Quotation to Sales Note:
   - Copy items
   - Adjust quantities/prices if needed
5. Record payment / credit.

## System requirements
- Separate folio sequences
- No retyping during conversion
